# TogetherView – Privacy Policy

*Last updated: March 15, 2026*

## What we collect

TogetherView does not collect, store, or transmit any personal data. The extension does not require an account, login, or any personally identifiable information.

## What we sync

When you are in an active watch party, TogetherView sends anonymous player events (play, pause, seek position in seconds) to a real-time relay server (Azure Web PubSub) solely to keep playback in sync between party members. These events contain no personal information and are not stored — they are relayed in real time and immediately discarded.

## What we store locally

TogetherView stores a randomly generated anonymous user ID in your browser's session storage. This ID is used only to prevent echo of your own player events. It is not linked to your identity and is cleared when your browser session ends.

## Third-party services

TogetherView uses Microsoft Azure Web PubSub solely as a real-time message relay. No data is persisted on Azure beyond the duration of a watch party session.
